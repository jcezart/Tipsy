# 🍹 Tipsy – Tip Calculator iOS App

Tipsy is a simple and intuitive iOS application that helps users calculate tips and split restaurant bills among multiple people.

The app allows users to quickly enter the bill amount, select a tip percentage, and divide the total evenly between friends.

This project was developed as part of my journey learning **native iOS development with Swift and UIKit**.

---

# 📱 Features

- Calculate tips (0%, 10%, 20%)
- Split the bill between multiple people
- Clean and simple user interface
- Instant calculation and result display
- Navigation between screens using Storyboards

---

# 🛠 Tech Stack

- Swift
- UIKit
- Storyboard
- MVC Architecture
- Auto Layout

---

# 🧠 What I Learned

Through this project I practiced and reinforced several important iOS development concepts:

- Building UI with **Storyboard**
- Handling **user input and data flow**
- Navigation between screens using **segues**
- Implementing the **MVC architecture pattern**
- Structuring logic inside **ViewControllers**
- Working with **Swift data types and calculations**

# 🚀 How to Run the Project

- Clone the repository: git clone https://github.com/jcezart/Tipsy.git
- Open the project: Tipsy.xcodeproj
- Run the app in the simulator: ⌘ + R
